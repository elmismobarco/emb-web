No estás solo.

Aunque a veces lo parezca, todos vamos en el mismo barco.

Este espacio es para eso. Para detenerte un poco. Para leer algo que, quizá, también te está pasando a ti.

No para tener respuestas. Sino para sentir que no eres el único haciéndote las preguntas.
